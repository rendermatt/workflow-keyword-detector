Base Fit Prompt

The single prompt sent to Claude for every assessment. It owns the placeholders, the scoring scale, and the output format. Per-feature nuance is added through each feature's Additional guidance field, which is injected at {% raw %}{{FEATURE_INSTRUCTIONS}}{% endraw %}.

These placeholders are filled in automatically before each request. Keep the ones you want the model to see.
{% raw %}
{{FEATURE_NAME}}
The feature's name.
{{FEATURE_DOCUMENTATION}}
Visible text from the feature's documentation URL — that one page — fetched once per crawl.
{{FEATURE_INSTRUCTIONS}}
This feature's Additional guidance field (a short note is inserted when a feature has none).
{{CUSTOMER_DOMAIN}}
The company's root domain being assessed.
{{CRAWLED_CONTENT}}
The combined visible text of every page crawled on that one company's site, each section headed by its URL — not a single page. Gathered from up to 50 pages and capped at ~60,000 characters total. One fit assessment is produced per company from this whole bundle (not one per page).
{% endraw %}
